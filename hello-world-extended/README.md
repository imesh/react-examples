
# Hello World Extended React.js Example

This is a hello world example written in React.js.

## How to Run

- Clone this git repository and switch to the `hello-world-extended` directory.

- Install the dependencies:
  
  ````bash
  npm install
  ````

- Run the application:
  
  ````bash
  npm start
  ````
